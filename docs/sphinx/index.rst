duvc-ctl Documentation
======================

duvc-ctl is a library for controlling USB Video Class (UVC) cameras on Windows using DirectShow APIs. It provides interfaces in C++, Python, and a command-line tool for device management, property control, and vendor-specific operations.

This documentation covers installation, usage, API references, and examples for developers working with the library.
